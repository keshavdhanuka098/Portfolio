# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

---

## Navigate

```bash
cd portfolio
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

## Build Project

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```
